## Запуск проекта: Frontend
#### 1. Перейдите в каталог `front`.
```
cd front
```
#### 2. Запустите сервер.
```
php -S localhost:7000
```

## Запуск проекта: Backend
#### 1. Перейдите в каталог `src`.
```
cd src
```
#### 2. Запустите создание миграций.
```
php cli.php migrate
```
#### 3. Запустите создание сидов.
```
php cli.php seed
```
#### 4. Запустите сервер.
```
php -S localhost:8000
```
